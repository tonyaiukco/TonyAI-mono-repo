# Product Overview: TonyAI Enterprise

## 1. Vision Statement
TonyAI is a high-end, enterprise-grade sustainability management platform designed for holding companies to track, calculate, and project carbon emissions across a complex structure of subsidiaries. It transforms raw activity data into audit-ready ESG intelligence.

## 2. Target Market & Compliance
- **Primary Users:** Sustainability Officers, ESG Consultants, and Corporate Auditors.
- **Reporting Standards:** ISO 14064-1, GRI 305, GHG Protocol, and UK SRS.
- **Regional Focus:** Initially optimized for the UK, Turkey (TR), and the European Union (EU).

## 3. Core Value Propositions
- **Granular Visibility:** A "Tracking Matrix" that identifies data gaps at the subsidiary level.
- **Dynamic Calculation:** Real-time emission results based on regional grid intensities and fuel types.
- **Audit Integrity:** Immutable data history with version control and evidence (invoice) linking.
- **Scalability:** Ability to manage hundreds of subsidiaries with role-based access.

## 4. Key Performance Indicators (KPIs)
The platform measures success through:
- **Calculation Completion Rate:** Percentage of required categories finalized.
- **Total Portfolio Footprint:** Real-time aggregated tCO2e.
- **Data Quality Score:** Ratio of metered vs. estimated data points.

## 5. Deployment Context
- **Infrastructure:** GCP / Azure (Containerized/Serverless Deployment for Next.js).
- **Architecture:** Headless UI connected to a calculation engine API via JSON.
